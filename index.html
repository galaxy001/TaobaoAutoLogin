TaobaoAutoLogin

Auto login safari extension for Taobao.
淘宝自动登录 Safari 扩展.

Intro | 简介

Set the username and password in Preferences - Extensions - TaobaoAutoLogin and it will do the rest for you.
在 偏好设置 - 扩展 - TaobaoAutoLogin 中设置好用户名和密码, 其余的登录过程会由插件自动完成.
